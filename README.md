# 🌐 IndiHome Yogyakarta — Landing Page

Landing page resmi untuk layanan pemasangan **IndiHome** wilayah **Yogyakarta**, dibangun menggunakan HTML, CSS, dan JavaScript tanpa framework tambahan untuk performa maksimal.

🔗 **Live Site:** [erman4u.github.io/Indihome-Purnama](https://erman4u.github.io/Indihome-Purnama/)

---

## ✨ Fitur Utama (v2.0 Upgrade)

- 💎 **Premium UI/UX Design** — Tampilan modern dengan efek *Glassmorphism*, gradien halus, dan tipografi Inter yang tajam.
- 🎨 **Lucide Icons Integration** — Mengganti semua emoji dengan library ikon open-source Lucide untuk tampilan yang lebih profesional dan bersih.
- ⚡ **Compact & Efficient Pricing** — Desain kartu paket yang telah dioptimalkan (lebih ramping) untuk efisiensi ruang dan kemudahan membaca di perangkat mobile.
- 🚀 **Smooth Animations** — Dilengkapi dengan *Scroll Reveal* dan animasi *micro-interaction* pada tombol dan kartu.
- 📱 **Fully Responsive** — Optimal di smartphone, tablet, maupun desktop.
- 📋 **Informasi Transparan** — Keterangan harga dasar (**Exclude PPN 11%**) sesuai regulasi terbaru.
- 💬 **WhatsApp Integrated** — Form cek jaringan dan tombol melayang yang langsung terhubung ke layanan pelanggan.
