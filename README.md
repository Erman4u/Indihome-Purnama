# 🌐 IndiHome Purnama — Landing Page

Landing page resmi untuk layanan pemasangan **IndiHome** wilayah **Yogyakarta**, dibangun menggunakan HTML, CSS, dan JavaScript murni tanpa framework tambahan.

🔗 **Live Site:** [erman4u.github.io/Indihome-Purnama](https://erman4u.github.io/Indihome-Purnama/)

---

## ✨ Fitur

- ⚡ **Tampilan Modern & Premium** — Desain responsif dengan font Inter, gradien halus, dan animasi micro-interaction
- 📱 **Responsif di Semua Device** — Mobile, tablet, dan desktop dengan hamburger menu untuk layar kecil
- 🔴 **Sticky Navbar** — Navbar transparan yang berubah warna saat scroll
- 💳 **Kartu Paket** — Menampilkan paket Internet Only, Internet + STB TV, dan Internet + Kuota
- 📋 **Cara Pasang** — Panduan 6 langkah pemasangan IndiHome
- ❓ **FAQ Accordion** — Tanya jawab interaktif dengan animasi smooth
- 📍 **Form Cek Jaringan** — Form langsung terhubung ke WhatsApp
- 💬 **Tombol WhatsApp Float** — Tombol "Daftar Sekarang" yang selalu terlihat dengan animasi pulse
